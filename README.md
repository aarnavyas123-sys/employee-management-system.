# Employee Management System

A full-stack Employee Management System built using React, Node.js, Express.js, and PostgreSQL.

## Features

### Authentication
- Login & Signup
- JWT Authentication
- Role-Based Access Control

### HR Module
- Add/Edit/Delete Employees
- Manage Departments
- Manage Skills
- Leave Approval Workflow
- Approval History

### Manager Module
- View Employees
- Approve/Reject Leave Requests
- View Approval History

### Employee Module
- View Profile
- Apply Leave
- Track Leave Status

## Tech Stack

Frontend:
- React.js
- Bootstrap
- React Router
- Recharts

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL
